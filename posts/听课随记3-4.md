# 损失函数 正则化

Why：**用训练集上一点点微小的误差，换取测试集上巨大的泛化能力**
$$
L(W) = \frac{1}{N} \sum_{i=1}^N L_i(f(x_i, W), y_i) + \lambda R(W)
$$
其中前半部分是**数据损失**，后半部分是正则化：

### L2正则化

$$
R(W) = \sum_k \sum_l W_{k,l}^2
$$

就是把权重矩阵的所有元素平方和，这会让接近0的数更小，其造成的代价也更小，所以L2正则化通常**会保留小的权重**。

**均匀地把所有权重都变小但不归零，更加分散**。

### L1正则化

$$
R(W) = \sum_k \sum_l |W_{k,l}|
$$

把权重矩阵的所有元素绝对值求和，比较小的数不会因为这个更小，但同时代价也不会变小，所以L1正则化**会把接近0的值变成0**，**会更稀疏**。

**把有用的权重保留，没用的变小成0**。

Why：总损失函数的目标就是最小化，在某项取正值但对模型性能无影响、前半部分不变的情况下，后半部分正则化项如果能尽量小，模型就会把其优化掉。

# 优化问题 Optimization

### 梯度下降

```python
while True:  # 可以是固定迭代次数，也可以是损失函数不再怎么下降的时候
    # 计算出权重矩阵尺寸大小的梯度
    weights_grad = evaluate_gradient(loss_fun, data, weights)  
    weights += - step_size * weights_grad
```

### 随机梯度下降 SGD

$$
x_{t+1} = x_t - \alpha \nabla f(x_t)
$$

```python
while True:
    data_batch = sample_training_data(data, 256)  # 每次随机挑出256个样本去做梯度下降
    weights_grad = evaluate_gradient(loss_fun, data, weights)  
    weights += - step_size * weights_grad
```

### SGD + 动量

$$
v_{t+1} = \rho v_t + \nabla f(x_t)\\
x_{t+1} = x_t - \alpha v_{t+1}
$$

加上动量之后，**鞍点**、**极小值**、**抖动**、**噪声的方向**等问题都会得到改善。

### RMSProp

```python
grad_squared = 0
while True:
    dx = compute_gradient(x)
    grad_squared = decay_rate * grad_squared + (1 - decay_rate) * dx * dx
    x -= learning_rate * dx / (np.sqrt(grad_squared) + 1e-7)
```

### Adam优化器

结合前面动量、RmsProp二者的思路，就可以得到Adam优化器：

```python
first_moment = 0
second_moment = 0

for t in range(1, num_iterations):
    dx = compute_gradient(x)
    # 更新一阶矩（即 Momentum：带方向的指数加权平均）
    first_moment = beta1 * first_moment + (1 - beta1) * dx
    # 更新二阶矩（即 RMSProp：梯度平方的指数加权平均，记录振幅/颠簸程度）
    second_moment = beta2 * second_moment + (1 - beta2) * dx * dx
    # 偏差修正（因为初始化为 0，早期迭代时估计值偏小，除以系数把它拉回正常大小）
    first_unbias = first_moment / (1 - beta1 ** t)
    second_unbias = second_moment / (1 - beta2 ** t)
    
    x -= learning_rate * first_unbias / (np.sqrt(second_unbias) + 1e-7)
```
